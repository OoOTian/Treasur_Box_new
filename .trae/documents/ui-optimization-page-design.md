# 页面设计规范（桌面端优先）

## 全局样式（Design Tokens）
- Layout: Flex + CSS Grid 混合；App Shell（侧边栏固定、内容区自适应）。
- 颜色：--bg:#F6F7FB；--surface:#FFFFFF；--text:#1F2329；--muted:#667085；--border:#E4E7EC；--primary:#3B82F6；--danger:#EF4444；--success:#22C55E；--warning:#F59E0B；--highlight-bg:#FEF3C7。
- 字体：系统字体栈；正文 14px/20px，标题 18/24、24/32；代码/预格式使用等宽字体。
- 圆角与阴影：radius 10；shadow: 0 8px 24px rgba(0,0,0,.08)。
- 间距：8/12/16/24 为基础；页面主容器 padding 24。
- 交互：按钮高度 32（small）/36（default）；hover 仅调整明度与阴影；focus 可见（outline 2px）。

## 1) 工具导航页（App Shell）
- Layout：左侧 Sidebar（宽 240，收起 72）+ 右侧 Content（flex:1，min-width:0）。
- Meta：title=“工具箱”；description=“文本处理小工具集合”。
- 结构：
  1. Sidebar Header：站点名 + 折叠按钮（单按钮切换，避免“展开/收起”双选）。
  2. Nav List：3 个入口（图标 + 文案）；选中态使用 --primary 浅底色 + 左侧标记条。
  3. Content Header：当前工具标题 + 简短说明（可选）。
  4. Content Body：路由渲染区域（禁止 iframe）。
- 响应：<1024 时 Sidebar 默认收起；内容区 padding 降为 16。

## 2) 限定词选取页
- Meta：title=“限定词选取”。
- 页面结构（Card 形式纵向堆叠）：
  1. 配置区：Checkbox（是否用 [ ] 包裹）。
  2. 输入区：Textarea（可自适应高度 4-8 行）+ 主按钮“提交”+ 次按钮“清空”。提交后输入区只读并显示“已提交，可清空后重新输入”。
  3. 选择区：大文本展示面板（等宽/可换行/可滚动，height: 240-320 自适应）；划取高亮使用背景色而非纯红字，保证可读性。
  4. 结果区：统计（数量/长度）+ 只读输入框展示结果 + 操作按钮组（撤销/清空/复制结果/复制完整内容）。
- 交互状态：空输入提交给出顶部 Message；复制成功/失败提示风格一致。

## 3) 文本分段+限定词选择页
- Meta：title=“文本分段 + 限定词选择”。
- 页面结构：
  1. Steps：两步；每步内容使用同一容器宽度与间距。
  2. Step1 输入：Textarea + 提交/清空。
  3. Step2 工具区：
     - 顶部：返回按钮 + “复制当前页文本”。
     - 分页：统一使用组件库 Pagination（替换自绘按钮），展示页码与快速跳转。
     - 配置：Checkbox（[ ] 包裹）。
     - 文本面板：pre/code 展示当前页文本并支持划取高亮。
     - 双栏结果：左“全文限定词”（只读 + “编辑”按钮打开弹窗）；右“本页划取限定词”（只读 + 撤销/清空）。
     - 统计：显示“划取数量/合并总长度”。
  4. 弹窗：列表编辑（输入框 + 否定勾选 + 删除）；底部展示拼接预览；确认时校验空项并聚焦到错误行。

## 4) OCR无关信息删除页
- Meta：title=“OCR无关信息删除”。
- 页面结构：
  1. 输入区：Textarea + “清空输入”。
  2. 删除项区：全选 Checkbox + 多选列表（换行布局，间距一致）+ “新增删除对象”按钮（弹窗）。
  3. 处理区：主按钮“处理文本”。
  4. 输出区：只读输出面板 + “复制输出内容” + 长度提示（>500 使用 warning/danger 色）。
- 弹窗：输入框 + 添加/取消；点击遮罩关闭。
